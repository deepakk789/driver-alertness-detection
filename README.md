# Driver Alertness Detection System

## Overview
This project is an AI-based system that detects driver drowsiness in real-time using a webcam.  
It analyzes eye closure and yawning patterns to determine whether the driver is alert or sleepy.

## Features
- Eye closure detection and Yawn detection using CNN model
- Real-time webcam monitoring using mediapipe
- Alarm system for drowsiness alert

## Tech Stack
- Python
- OpenCV
- Mediapipe
- TensorFlow / Keras
- NumPy, SciPy
